# uptex-texmf

The bundle contains the base files for [upTeX by Takuji Tanaka](http://www.t-lab.opal.ne.jp/tex/uptex_en.html).
This is a community edition syncing with [ptex-texmf tree](https://github.com/texjporg/ptex-texmf).
